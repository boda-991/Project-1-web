Senior Frontend Developer

Google Inc.

Full Time

2026/8/12

Job Description:

We are looking for a Frontend Developer to join our team

Key Responsibilities

  • Develop User Interfaces: Create, maintain, and modify web application front-end code using HTML, CSS, and JavaScript/TypeScript.
  • Implement Designs: Translate UI/UX design wireframes into high-quality, functional, and responsive code.
  • Optimize Performance: Optimize web applications for maximum speed, scalability, and performance across various browsers and devices
  • Collaboration: Work closely with backend developers, UI/UX designers, and product managers to ensure a cohesive user experience.
  • Testing and Debugging: Identify and resolve bugs, issues, and browser compatibility issues.

Required Skills and Qualifications

  • Technical Proficiency: Expert-level knowledge of HTML5, CSS3, and JavaScript.
  • Frameworks: Experience with modern front-end frameworks and libraries such as React.js, Angular, or Vue.js.
  • Responsive Design: Deep understanding of mobile-first, responsive design principles.
  • Tools: Familiarity with version control systems (e.g., Git), build tools (e.g., Webpack), and browser developer tools.
  • Concepts: Knowledge of web accessibility standards (WCAG) and API integration.
